### 📖 使用说明

#### 1. 软件概述
本工具旨在提高从 Excel 导入数据到 CATIA 工程图（.CATDrawing）的效率。
* **局限性**：复杂的合并单元格及特殊划线可能无法完美还原。
* **字符显示**：若转入后字符不显示，请在 CATIA 中尝试更换字体（如 SimSum 或 Arial）。
* **技术背景**：基于 Python 编写，封装完整运行环境，体积约 40MB+。

#### 2. 运行前提
* **操作系统**：Windows 10 及以上。
* **CATIA**：支持 V5 R19 及以上版本。运行时需先启动 CATIA 并打开一张图纸。
* **Excel**：支持 Office 2013 及以上。**注意**：仅安装 WPS 可能会导致连接失败。

#### 3. 操作步骤
1. **启动软件**：确保目标 CATIA 图纸开启，并在 Excel 中选中目标数据区域。
2. **运行工具**：双击运行 `Excel_Converter.exe`。
3. **执行导入**：
    * 确认下拉框已识别图纸。
    * 点击 **“2. 检测 Excel 选区范围”** 核对行列数。
    * 点击 **“3. 确认无误，开始导入”**，等待进度条完成。
